Gallery Page

Technical Description

JavaScript is used to open a modal when a gallery item is clicked, dynamically updating image, title, and description using data attributes. It also handles modal closing and toggling font style and color theme.

Accessibility

Content Page

Technical Description

CSS is used to style the article layout with centered content, box shadows, and responsive images, while Flexbox is used to align images side by side. Additional styling like borders, spacing, and a fixed “go to top” button improves usability.

Accessibility

Images include alt text to describe them for users. Simple language and clear headings make the content easy to understand. The page layout is clean and helps users read and navigate easily.
